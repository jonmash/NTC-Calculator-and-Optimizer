Notes

  • NTC model uses the Beta equation: R(T) = R0 · exp[ β·(1/T − 1/298.15) ], with T in Kelvin and R0 at 25 °C.
  • Divider assumed as RS to VS, NTC to GND, Vout at the midpoint: Vout = VS · RNTC/(RS+RNTC).
  • ADC code ≈ round( Vout/VREF · (2N−1) ). If your ADC reference ≠ VS, set VREF accordingly.
  • Standard E12 resistor series is used when suggesting practical RS values.
